## List type

You can specify the type of list the item is in by passing the correct value `type` prop.
However, if you are using this component from within a `List` component then it will automatically
inherit the type from the parent.

## Interactions

You can add an `onClick` handler to a `ListItem` to make it interactive. This will add a hover
state and a focus state to the item.

<Canvas withToolbar sourceState="shown" of={Stories.IsClickable} />
